# Weather App - Aplicación del Clima

Este es un proyecto de una Aplicación del Clima interactiva y responsiva. Permite a los usuarios buscar el estado del tiempo actual de cualquier ciudad del mundo en tiempo real, utilizando datos meteorológicos oficiales

[cite_start]Este proyecto fue diseñado específicamente para demostrar habilidades sólidas en el desarrollo Frontend y el consumo de servicios web externos.

**[Haz clic aquí para ver la Demo en Vivo]([https://weather-app-zamarripa.netlify.app/])**

---

## Tecnologías Utilizadas

* [cite_start]**HTML5:** Estructura semántica del DOM.
* [cite_start]**CSS3:** Diseño responsivo, efectos de desenfoque (*glassmorphism*) y transiciones dinámicas.
* [cite_start]**JavaScript (Vanilla):** Lógica de programación, manipulación del DOM y eventos.
* [cite_start]**Fetch API:** Consumo de datos asíncronos.
* [cite_start]**OpenWeatherMap API:** Proveedor oficial de los datos climáticos en tiempo real.

---
* [cite_start]**Manejo de Promesas (Async/Await):** Implementación de flujos asíncronos limpios para consultar la API de forma eficiente.
* [cite_start]**Control de Errores (Try/Catch):** Manejo preventivo de fallos en la red o búsquedas de ciudades inexistentes, mostrando mensajes amigables al usuario en lugar de romper la app.
* [cite_start]**Manipulación Dinámica del DOM:** Actualización en tiempo real del texto, imágenes y estructuras del HTML según la respuesta del servidor.
* [cite_start]**Toque Pro - Reactividad Visual:** El fondo de la pantalla cambia por completo (mediante inyección de clases CSS) dependiendo de si el cielo está despejado, nublado o lluvioso en la ciudad consultada.
